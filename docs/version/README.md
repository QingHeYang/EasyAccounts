## 版本命名规则

- **主版本号**：重大架构变更或不兼容更新
- **次版本号**：新增功能或重要特性
- **修订版本号**：Bug 修复和小改进

## 更新建议

- 建议始终使用最新版本以获得最佳体验
- 重大版本更新前请仔细阅读升级说明
- 升级前务必备份数据库
